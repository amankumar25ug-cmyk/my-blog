Unit 2  In this unit, I explore new and many interesting things related to digital images. At first I felt very confused but slowly I started understanding them better.  One of the important things I learned is how images are represented using bits. Colours like black and white images use very simple representations, where each pixel is either 1 or 0. But this is not same for all like coloured images are more complex because they use more bits to represent different colours. I found it interesting how a simple image is made with so much data. After that I learned about pixels, resolution,and how image quality depends on these factors. Before this unit I had no idea how these images were formed or stored. But now I am able to understand that every image we see on screen is actually a combination of many pixels.Another important topic was  colour representation. We discuss all about how coloured are stored and how editing tools can change brightness, contrast and other properties. This is very useful for me to understand how photo editing works in real life. We also have some practical work for these topics. These practicals made learning more interesting. At first,I found it a bit difficult to follow the steps, especially when using new tools and commands. But when I tried many times or did practice I became more comfortable. Doing hands-on work helped me understand the concepts better. Overall, this unit was very useful and engaging. It helped me see technology from a different perspective. Now with the help of this unit I can understand how digital images work, not just use them.I believe this knowledge will help me for future studies and projects.
